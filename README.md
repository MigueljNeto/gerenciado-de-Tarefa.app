<h1>O gerenciamento do estado das tarefas é feito pelo StatefulWidget, enquanto os campos de entrada utilizam o TextEditingController, o que possibilita a captura reativa do texto inserido pelo usuário.</h1> 
<h1>Ademais, as tarefas são organizadas automaticamente: as pendentes aparecem primeiro, seguidas das concluídas, e os títulos são ordenados em ordem alfabética.</h1>
<h1>Para incluir uma nova tarefa, o aplicativo apresenta um AlertDialog com um campo de texto; ao confirmar, a tarefa é adicionada à lista e a interface é atualizada.</h1>
<h1>Esta tela ilustra claramente a modularização do código, a distinção entre lógica e interface, além de empregar as estruturas nativas do Flutter para proporcionar uma experiência funcional e intuitiva.</h1>
